# Saudi Arabia Leads Sniper & Validator

Automated Lead Generation system designed to scrape, verify, and identify decision-makers for companies in Saudi Arabia.

## 🚀 Features
- **Module 1 & 2**: Google Maps & Website Scraper using Selenium.
- **Module 3 (The Validator)**: DNS MX-Record verification to ensure 100% email deliverability.
- **Module 4 (Identity Sniper)**: OSINT-based identity matching using Bing/Google Dorking to find Managers/CEOs.
- **Anti-Captcha**: Implemented human-mimicry delays and DuckDuckGo/Bing fallback to bypass bot detection.

## 🛠️ Tech Stack
- Python 3.x
- Selenium (Automation)
- Pandas (Data Processing)
- DNSPython (Email Validation)

## 📖 How to Use
1. Open Chrome in remote debugging mode:
   `chrome.exe --remote-debugging-port=9222`
2. Run the modules in sequence (1 to 5).
